# Database Migrations

This directory contains Drizzle ORM migration files generated by `drizzle-kit`.

## Commands

```bash
# Generate migration SQL from schema changes
npm run db:generate -w packages/db

# Apply pending migrations to the database
npm run db:migrate -w packages/db

# Push schema directly (development only — no migration file)
npm run db:push -w packages/db

# Open Drizzle Studio (visual DB browser)
npm run db:studio -w packages/db
```

## Workflow

1. Modify schema files in `packages/db/src/schema/`
2. Run `npm run db:generate -w packages/db` to generate SQL migration
3. Review the generated SQL in this directory
4. Run `npm run db:migrate -w packages/db` to apply
5. Commit the migration files to version control

> **Never use `db:push` in production.** It bypasses the migration system and can cause data loss.
