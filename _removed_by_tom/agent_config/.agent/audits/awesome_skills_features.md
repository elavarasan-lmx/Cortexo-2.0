## What It Is

The **largest installable library of AI agent skills** in the world. It packages 1,431+ reusable `SKILL.md` playbooks for Claude Code, Cursor, Codex CLI, Gemini CLI, Antigravity, Kiro, OpenCode, and GitHub Copilot. It includes an npm CLI installer, role-based bundles, execution workflows, a web catalog app, and automated skill management tooling.

## Key Architectural Features for Cortexo Comparison

1. **Massive Skill Library (1,431+)** — Cortexo's F117 (Fractal Skill Library) exists but at a much smaller scale.
2. **Risk Classification System** — Every skill is tagged by risk level. Cortexo has no equivalent.
3. **Bundle System** — Role-based skill bundles (Web Wizard, Security Engineer, etc.) for quick onboarding.
4. **Workflow Orchestration** — Ordered execution playbooks that chain skills together.
5. **Cross-Tool Compatibility** — Skills work across 10+ AI coding assistants.
6. **Marketing/Growth Skills** — 80+ marketing skills that Cortexo doesn't cover at all (SEO, CRO, Growth).
7. **100+ Integration Automations** — Ready-made skills for Slack, Jira, Salesforce, HubSpot, etc.
8. **Supply Chain/Logistics Skills** — Industry-specific vertical skills.
9. **Agent Overload Recovery** — Guidance for context-limit management when too many skills are active.
10. **Security Risk Guardrails** — Offensive skills are labeled and require explicit authorization.
